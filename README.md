# 📊 Análise Comercial e Operacional – Sample Superstore

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de analisar o desempenho comercial e operacional da base de dados **Sample Superstore**, utilizando consultas em **SQLite** para exploração e análise dos dados e **Power BI** para criação de dashboards interativos.

O projeto simula um cenário real de análise de dados, onde informações de vendas, lucro, clientes, produtos e logística são transformadas em insights para apoiar a tomada de decisão.

---

# 🎯 Objetivos

* Analisar o desempenho de vendas e lucratividade.
* Identificar as categorias e produtos com melhor desempenho.
* Avaliar o comportamento das vendas por região.
* Analisar indicadores operacionais relacionados aos pedidos.
* Desenvolver um dashboard executivo para facilitar a visualização dos resultados.
